# customer-api
This API can be used by resellers' systems to create or retrieve customer data in Fuank's CRM.

Following functionalities are supported by this API:

### 1. Create Customer
```bash 
POST /customers
```

### 2. Retrieve Customer
```bash 
GET /customers/{customerId}
```

### 3. Update Customer
```bash 
PATCH /customers/{customerId}
```
